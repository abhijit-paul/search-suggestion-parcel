# Search suggestion utility
A simple search suggestion utility

# What is search suggestion?
The search application is a word suggestion app, where as you keep typing words, you get a set of suggestions. You can navigate through this list of suggestions and then select one using keyboard up/down buttons. 

#Details
At every keyup event on the search box, a call is made to the suggestions are fetched and the list of suggestions are updated asynchronously. A suggestion can be selected either by navigating to it using keyboardâ€™s down/up keys and then selected using the Enter button; or can be directly chosen with a mouse click.
